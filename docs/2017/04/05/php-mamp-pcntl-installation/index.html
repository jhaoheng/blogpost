【PHP-MAMP】pcntl 安裝

Posted on 2017-04-05

環境

  • osx
  • php7.0.8
  • MAMP 3.5.2

flow

  1. brew search pcntl，找尋要安裝的版本
  2. brew install php70-pcntl，得到 pcntl 的安裝位置 => pcntl__path
  3. 安裝完畢後，確認 php, which php，得到路徑 => php__path
  4. ls -al 路徑(php__path)，確認 php 的 source 指到的是 MAMP 的 php 路徑
  5. 在 pcntl__path 中，取得 pcntl.so 的檔案，copy 到 MAMP 的 extension 中
  6. 編輯 localhost 的 php.ini 與 MAMP 的 php.ini，加入 extension=pcntl.so
  7. 在 terminal 中，php -i | grep pcntl，可看到版本訊息
