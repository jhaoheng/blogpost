Posted on 2017-05-18 | In HLS , AWS

環境

  • player : http://video-dev.github.io/hls.js/demo/
  • aws s3
    • 將 ts 與 m3u8 均放在上面

問題描述

  1. VLC 可播放
  2. 丟入其他播放器，無法播放
    • 某些播放器會說明 crossdomain access denied
    • 持續轉圈圈

解決 crossdomain access denied

  • aws file : http://docs.aws.amazon.com/AmazonS3/latest/dev/cors.html
  • console 設定
    • 進入 bucket
    • 進入所選定的 folder 中
    • 選擇 Permissions
    • 看到 AllowedHeader
    • 寫入 *
